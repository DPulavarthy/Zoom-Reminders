## How to use

❯ Clone the repo.

```
git clone https://github.com/DPulavarthy/Zoom-Reminders.git
```
❯ Install the modules.
```
npm i
```

❯ Add the links

Open `links.js` and put the zoom links in the according field.

❯ Run the code.
```
npm start
```

Note: Do not close the terminal.
